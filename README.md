# database-test
Приложение состоит из 3 компонент: фронтенд JS (Vue 3), бэкенд Python (Flask), БД PostgreSQL. Предварительно запустите серверную часть 
```
python app.py
```
используйте `migrate-db.py` (расположен в корне проекта) чтобы накатить миграцию БД.

Далее используйте предложенные команды, чтобы запустить фронтенд

## Project setup
```
npm install
```
